# Rocky Fitness Center

Welcome to the **Rocky Fitness Center** repository! 

This is the source code for the official website of Rocky Fitness Center, a modern gym and fitness facility. The website is built with HTML, CSS, and JavaScript, designed to showcase our facilities, classes, trainers, and membership plans.

## 🌐 Live Demo

Check out the live website here: **[Rocky Fitness Center](https://rockyfitnesscenter.in)**

## 🏋️ Features

- **Home Page**: A welcoming landing page highlighting the gym's core offerings.
- **Classes**: Information about various fitness programs and group classes.
- **Trainers**: Meet our expert trainers and fitness coaches.
- **Membership**: Details on pricing, subscription tiers, and gym passes.
- **Achievements**: Success stories and milestones from our gym members.
- **Blog**: Fitness tips, health guides, and news.
- **Contact**: Get in touch, find our location, and view our operating hours.

## 📁 Repository Structure

- `index.html` - Main landing page
- `classes.html` - Fitness classes schedule and details
- `trainers.html` - Profiles of fitness instructors
- `membership.html` - Pricing and membership plans
- `achievements.html` - Gym and member achievements
- `blog.html` - Fitness articles and news
- `contact.html` - Contact form and location details
- `css/` - Stylesheets for layout and design
- `js/` - JavaScript files for interactive elements
- `assets/` - Icons, fonts, and other static assets
- `*.jpg` / `*.jpeg` - Image assets used across the site

## 🚀 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your browser:

```bash
# Clone the repository
git clone https://github.com/Zyvrix/rocky-fitness.git

# Navigate to the project folder
cd rocky-fitness

# Open the website (e.g., on macOS)
open index.html
```

## 🛠️ Technologies Used

- **HTML5** for structure
- **CSS3** for styling and responsiveness
- **JavaScript** for interactivity
- **Vercel** for hosting/deployment
